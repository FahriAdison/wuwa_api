Introduction

Welcome to the Wuthering Waves API documentation. This API provides access to comprehensive data about the Wuthering Waves game, including characters, weapons, echoes, and more. The API is RESTful and returns data in JSON format.

GET /api

Returns basic information about the API, including version and available endpoints.

Example Response

{
  "status": 200,
  "message": "Success",
  "data": {
    "message": "Welcome to Wuthering Waves API (JavaScript Version)",
    "version": "2.2",
    "documentation": "https://github.com/FahriAdison/wuwa_api",
    "endpoints": [
      "/api/characters",

      "/api/echoes",
      "/api/attributes",
      "/api/sonatas",
      "/api/stats",
      "/api/substats",

  },
  "timestamp": "2025-04-22T11:20:00.000Z"
}

Authentication

The Wuthering Waves API is currently open and does not require authentication. However, please respect the rate limits to ensure fair usage for all users.

Rate Limiting

To ensure fair usage, the API implements rate limiting. Each IP address is limited to 100 requests per 15-minute window. If you exceed this limit, you will receive a 429 Too Many Requests response.

Rate Limit Exceeded Response

{
  "status": 429,
  "error": true,
  "message": "Too many requests from this IP, please try again after 15 minutes",
  "timestamp": "2025-04-22T11:20:00.000Z"
}

Characters

Access information about characters in Wuthering Waves, including their stats, abilities, and more.

GET /api/characters

Returns a list of all characters in Wuthering Waves.

Query Parameters

Parameter Type Description
page integer Page number for pagination (default: 1)
limit integer Number of items per page (default: 20, max: 100)
