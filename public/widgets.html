API Features

Explore the available endpoints and features

  • Characters
    /characters
    Get a list of all characters in Wuthering Waves.
    /characters/:id
    Get detailed information about a specific character.
    Example Response
    {
  "id": "cantarella",
  "name": "Cantarella",
  "element": "Electro",
  "weapon": "Pistol",
  "rarity": 5,
  "stats": {
    "attack": 845,
    "defense": 520,
    "hp": 12450
  }
}
  • Weapons
    /weapons
    Get a list of all weapons in Wuthering Waves.
    /weapons/:id
    Get detailed information about a specific weapon.
    Example Response
    {
  "id": "the_last_dance",
  "name": "The Last Dance",
  "type": "Pistol",
  "rarity": 5,
  "stats": {
    "attack": 500,
    "secondary_stat": "CRIT Rate",
    "secondary_value": 22.1
  }
}
  • Echoes
    /echoes
    Get a list of all echoes in Wuthering Waves.
    /echoes/:id
    Get detailed information about a specific echo.
    Example Response
    {
  "id": "virtuoso_sonata",
  "name": "Virtuoso Sonata",
  "rarity": 5,
  "effect": "Increases ATK by 30% and CRIT DMG by 15%"
}
  • Attributes
    /attributes
    Get a list of all attributes in Wuthering Waves.
    /attributes/:id
    Get detailed information about a specific attribute.
  • Sonatas
    /sonatas
    Get a list of all sonatas in Wuthering Waves.
    /sonatas/:id
    Get detailed information about a specific sonata.
  • Stats
    /stats
    Get a list of all stats in Wuthering Waves.
    /stats/:id
    Get detailed information about a specific stat.
  • Substats
    /substats
    Get a list of all substats in Wuthering Waves.
    /substats/:id
    Get detailed information about a specific substat.
  • Codes
    /codes
    Get a list of all redemption codes in Wuthering Waves.
    /codes/:id
    Get detailed information about a specific redemption code.
